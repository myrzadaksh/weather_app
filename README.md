# Weather App

A small app to view current weather, forecast, air quality, and map layers.

**Features**
- current weather + local time
- hourly and daily forecast
- extra info (UV, wind, pressure, sunrise/sunset)
- map with weather layers and map type switch
- air quality index (AQI)

**Tech Stack**
React + TypeScript + Vite + TanStack Query + Zod + Leaflet

**Run locally**
```bash
npm install
npm run dev
